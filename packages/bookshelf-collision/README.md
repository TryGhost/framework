# Bookshelf Collision

## Install

`npm install @tryghost/bookshelf-collision --save`

or

`pnpm add @tryghost/bookshelf-collision`

## Purpose

Bookshelf plugin that detects stale updates and throws an update-collision error when client and server versions diverge.

## Usage

## Develop

This is a mono repository, managed with [Nx](https://nx.dev).

Follow the instructions for the top-level repo.

1. `git clone` this repo & `cd` into it as usual
2. Run `pnpm install` to install top-level dependencies.

## Run

- `pnpm dev`

## Test

- `pnpm lint` runs oxlint
- `pnpm test` runs lint and tests
